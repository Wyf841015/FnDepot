# USB Rsync - USB自动同步工具

![Version](https://img.shields.io/badge/version-v0.1.7-blue)
![Platform](https://img.shields.io/badge/platform-Linux%20%7C%20fnOS-green)
![License](https://img.shields.io/badge/license-MIT-orange)

USB存储设备自动同步工具，检测到USB设备插入后自动同步指定目录。

## 功能特性

### 🔄 同步功能
- **单向同步** - 从源目录同步到目标目录
- **双向同步** - 源目录和目标目录双向同步
- **插入自动同步** - 检测到USB设备插入时自动执行同步任务
- **手动同步** - 随时手动触发同步任务

### 📊 设备管理
- 实时检测USB设备插入/移除
- 显示设备名称、序列号、文件系统、容量
- 支持热插拔

### 💾 任务管理
- 创建多个同步任务
- 独立保存每个任务
- 启用/禁用任务
- 实时显示同步状态

### 📝 日志功能
- 实时日志刷新
- 日志自动刷新开关
- 同步历史记录

### ❤️ 赞助支持
- 内置赞助弹窗
- 支付宝/微信二维码
- 图片点击放大

## 安装

### FPK安装包
1. 下载 `USBRsync.fpk`
2. 在飞牛NAS应用中心安装

### 命令行安装
```bash
# 解压FPK
tar -xzf USBRsync.fpk

# 运行安装脚本
./cmd/install_init
./cmd/install_callback
```

## 配置说明

### 同步任务配置

| 参数 | 说明 |
|------|------|
| 任务名称 | 任务标识名称 |
| 源目录 | 同步来源目录 |
| 目标目录 | 同步目标目录 |
| 同步模式 | 单向同步(→) / 双向同步(⇄) |
| USB自动同步 | 插入USB时自动执行 |

### 路径变量

同步目标目录可使用以下变量：

| 变量 | 说明 |
|------|------|
| `{uuid}` | 设备UUID |
| `{label}` | 设备标签 |
| `{device}` | 设备名称 |
| `{mount_point}` | 挂载点 |

示例: `/mnt/usb0/backup_{uuid}` → `/mnt/usb0/backup_12345678`

## 常见问题

### Q: 同步失败怎么办？
A: 检查日志窗口的错误信息，常见原因：
- 源目录不存在
- 目标目录无写入权限
- rsync未安装

### Q: USB设备未检测到？
A: 
- 确认USB设备已正确挂载
- 检查设备文件系统是否支持

### Q: 如何查看同步进度？
A: 点击任务卡片的"立即同步"按钮，日志窗口会显示rsync的实时进度。
