## 安装说明

### 日志哨兵 (fnlogpush)

飞牛NAS日志监控与多渠道推送工具

**下载地址**: [fnlogpush.fpk](fnlogpush/fnlogpush.fpk)

**功能**：
- 实时监控系统日志和备份进度
- 支持企业微信、钉钉、飞书、Bark、PushPlus、MeoW等推送渠道
- Web界面管理、事件订阅、免打扰和告警聚合
- **事件管理功能** - 添加/编辑/删除自定义事件
- **事件查询** - 可查询数据库中事件的最新记录
- **5分钟无操作自动退出登录**

**界面特性**：
- 6个精选主题（暗夜紫/深色/深海蓝/清新绿/暮色橙/科技感霓虹）
- 加载动画、骨架屏、卡片悬浮动效
- 科技感霓虹主题（Cyber）
- 状态指示器脉冲发光
- 自定义滚动条、输入框增强效果

**更新日志 (v0.9.9)**：
- 同步至最新代码版本

**更新日志 (v0.9.5)**：
- 丰富推送信息：相册照片数量、相册名称、备份路径等
- 分享相册时只推送分享消息，不推送新建相册消息
- 点击备份卡片保存按钮时自动启动/停止备份监控
- 系统状态卡片优化：数据库和监控状态同行动态
- 删除系统状态卡片的重置监控按钮

### USB自动同步 (usbrsync)

USB存储设备自动同步工具，检测到USB设备插入后自动同步指定目录。

**下载地址**: [USBRsync.fpk](usbrsync/USBRsync.fpk)

**功能**：
- 单向同步 - 从源目录同步到目标目录
- 双向同步 - 源目录和目标目录双向同步
- 插入自动同步 - 检测到USB设备插入时自动执行同步任务
- 手动同步 - 随时手动触发同步任务
- 实时显示同步进度

**界面特性**：
- 全新深色主题设计（渐变背景 + 毛玻璃效果）
- 响应式布局（自动适配桌面端和移动端）
- 流畅动画（卡片淡入、悬停变换）
- 实时日志刷新

### 清理精灵CGI (fnclearupcgi)

扫描FnOS所有vol目录，找出已安装应用但残留目录文件的孤立应用，一键清理。

**下载地址**: [fnclearup.fpk](fnclearupcgi/fnclearup.fpk)

**功能**：
- 自动扫描 `/mnt/vol*` 下所有 `@app*` 目录
- 与已安装应用列表对比，找出孤立目录
- 支持多选批量删除，可预览完整路径
- 直接打开即可使用，无需注册或登录

**界面特性**：
- Tab 切换：网盘挂载目录 / data/vol02 未挂载目录独立展示
- KPI 卡片：显示已挂载数量和未挂载数量
- 主题按钮与赞助按钮风格统一
- 响应式布局，适配桌面端和移动端
- 纯 Bash CGI 实现，轻量无依赖

**更新日志 (v0.2.4)**：
- **修复**: 用 `docker inspect` 精确判断卷是否被容器引用，移除不可靠的 `used_file` grep 逻辑

**更新日志 (v0.2.3)**：
- 新增 Docker 卷管理功能（查看在用卷/残余卷，一键删除）

**更新日志 (v0.2.0)**：
- 新增 Tab 切换：网盘挂载目录 / data/vol02 未挂载目录独立展示
- 新增 KPI 卡片：显示已挂载数量和未挂载数量
- 主题按钮样式优化：与赞助按钮风格统一
- CSS 修复：`html.dark` 选择器、重复残缺样式块清理
- JS 修复：App 对象重复属性、`switchTab` 添加 `async` 关键字

**更新日志 (v0.1.5)**：
- 前端审计修复: XSS防护(innerHTML转义)、label for属性、WCAG AA对比度优化
