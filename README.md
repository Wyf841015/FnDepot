# FnDepot 飞牛NAS第三方应用仓库

> 收录飞牛NAS第三方应用，提供一站式下载和更新

## 应用列表

### FNLogPush 日志推送

- **版本**: 0.5.0
- **描述**: 自动监控飞牛NAS系统日志和备份进度，实时推送至多种渠道
- **标签**: 工具, 效率, 监控

## 使用方法

### 飞牛NAS应用商店添加

在飞牛NAS应用商店中添加第三方仓库：

```
https://raw.githubusercontent.com/Wyf841015/FnDepot/main/fnpack.json
```

## 项目展示

### FNLogPush 功能

- 📊 日志监控 - 实时监控系统日志
- 💾 备份监控 - 跟踪备份进度
- 📢 多渠道推送 - 企业微信、钉钉、飞书、Bark等
- 🔔 告警聚合 - 智能合并重复告警
- 🎨 多主题 - 支持多种界面主题
